# Bansi

Backend del proyecto Bansi hecho en ASP.NET Core con SQL Server y Swagger.

## Tecnologías usadas

- ASP.NET Core
- C#
- SQL Server
- Swagger
- Stored Procedures

## Funcionalidades

- Consulta de registros
- Alta de registros
- Actualización de registros
- Eliminación de registros
- Conexión a SQL Server mediante API REST
